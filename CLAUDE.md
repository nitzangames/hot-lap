## Project

2D Racing Game built with JavaScript/HTML5 Canvas. Part of the JSGames collection.

## Development

Open `index.html` in a browser to run the game. No build step or package manager required.

## Architecture

This is a vanilla JS browser game — no frameworks or bundlers. Game logic runs in a single-page HTML5 Canvas application.
